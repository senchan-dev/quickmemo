# Quick Memo

Edgeサイドバーやブラウザで使える、軽量・保存なしの一時メモページ。  
このコードは Microsoft Copilot によって生成されました。

## 特徴

- ダークモード固定（軽快で目に優しい）
- 文字数表示（改行は除外）
- コピー＆削除ボタン（Material Icons使用）
- GitHubリンク付き（ソース確認用）
- 保存機能なし（localStorageやcookie未使用）
- フォントサイズ固定（変更機能なし）

## 使い方

- メモを入力
- コピー：ボタンまたは `Cmd+Enter`
- 削除：ボタンで一括クリア

## 方針

- 高機能化しない
- 軽快さと信頼性を重視
- 必要ならローカルでテストしてから反映
