# Quick Memo

Edgeサイドバー用の軽量メモページ。保存されず、即入力・即コピーが可能。

## 特徴

- ダークモード固定（軽快で目に優しい）
- 文字数表示（改行除外）
- コピー＆削除ボタン（Material Icons使用）
- GitHubリンク付き（ソース確認用）
- 保存機能なし（localStorageやcookie未使用）
- フォントサイズ固定（変更機能なし）

## 使い方

1. GitHub Pagesで公開
2. EdgeのサイドバーにURLを追加
3. メモを入力
4. コピー：ボタンまたは `Cmd+Enter`
5. 削除：ボタンで一括クリア

## 方針

- 高機能化しない
- 軽快さと信頼性を重視
- 必要ならローカルでテストしてから反映
